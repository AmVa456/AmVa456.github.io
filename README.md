## Setup

This site uses GitHub Pages to host project documentation and demos. Each project has its own subdirectory.

To add a new project:
1. Create a subdirectory with the project name
2. Add an `index.html` file in that subdirectory
3. Update the main `index.html` to link to the new project

## Local Development

You can test this site locally by running a simple HTTP server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
