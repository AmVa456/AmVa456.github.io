Bot Commands

Use these slash commands to interact with xtab-pro

/project create Project

Create a new project with name and description

/project status Project

Check the status of your current project

/task add Tasks

Add a new task to the current project

/task list Tasks

List all tasks with their current status

/code share Code

Share a code snippet with syntax highlighting

/git status Git

Check the status of linked Git repository

/help Info

Show all available commands and documentation

/dashboard Info

Display project dashboard with overview

Getting Started

Add xtab-pro to your Discord server in just a few steps

1

Prerequisites

You need administrator permissions on your Discord server to add bots.

2

Invite the Bot

Click the "Add to Discord" button and select your server from the dropdown menu.

3

Authorize Permissions

Review and authorize the required permissions for xtab-pro to function properly.

4

Start Using

Type /help in any channel to see all available commands and get started!
